This section covers the supported settings for the method [Folder.AnalyzeForSlate(markerColor)](../resolve_api/Folder.md#analyzeforslatemarkercolor) and [MediaPoolItem.AnalyzeForSlate(markerColor)](../resolve_api/MediaPoolItem.md#analyzeforslatemarkercolor)

`markerColor` can be one of the following constants:

```jsx
resolve.MARKER_BLUE
resolve.MARKER_CYAN
resolve.MARKER_GREEN
resolve.MARKER_YELLOW
resolve.MARKER_RED
resolve.MARKER_PINK
resolve.MARKER_PURPLE
resolve.MARKER_FUCHSIA
resolve.MARKER_ROSE
resolve.MARKER_LAVENDER
resolve.MARKER_SKY
resolve.MARKER_MINT
resolve.MARKER_LEMON
resolve.MARKER_SAND
resolve.MARKER_COCOA
resolve.MARKER_CREAM
```
